pyLOM.DMD.run(X, r, remove_mean=True)#

Run DMD analysis of a matrix X.

Inputs:

  • X[ndims*nmesh,n_temp_snapshots]: data matrix

  • remove_mean: whether or not to remove the mean flow

  • r: maximum truncation residual

Returns:

DMD Modes - muReal: Real part of the eigenvalues - muImag: Imaginary part of the eigenvalues - b: Amplitude of the DMD modes - Variables needed to reconstruct flow

Return type:

  • Phi
